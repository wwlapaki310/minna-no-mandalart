# みんなのマンダラート 🌸

> シンプルに作成・投稿・閲覧できるマンダラートWebアプリ

## 📝 概要

マンダラート(曼荼羅図)は、目標達成や思考整理のための9x9のフレームワークです。
「みんなのマンダラート」では、自分のマンダラートを作成し、他の人と共有できます。

## 📖 マンダラートとは？

マンダラートは、メジャーリーガーの大谷翔平選手も高校時代に使用していたことで有名な目標設定・思考整理の手法です。

### 構造
```
┌─────────┬─────────┬─────────┐
│ 中目標1 │ 中目標2 │ 中目標3 │
│ の      │ の      │ の      │
│個別目標 │個別目標 │個別目標 │
├─────────┼─────────┼─────────┤
│ 中目標4 │【中央】 │ 中目標5 │
│ の      │ 大目標  │ の      │
│個別目標 │+中目標  │個別目標 │
├─────────┼─────────┼─────────┤
│ 中目標6 │ 中目標7 │ 中目標8 │
│ の      │ の      │ の      │
│個別目標 │個別目標 │個別目標 │
└─────────┴─────────┴─────────┘
```

1. **中央に「大目標」**を配置（最終的に達成したいこと）
2. **周囲8マスに「中目標」**を配置（大目標達成のための要素）
3. **それぞれの中目標を中心とした8マスに「個別目標」**を配置（具体的な行動）

合計81マス(9x9)で、目標達成のための具体的なアクションプランを可視化できます。
