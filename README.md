# みんなのマンダラート 🌸

> シンプルに作成・投稿・閲覧できるマンダラートWebアプリ

## 📝 概要

マンダラート(曼荼羅図)は、目標達成や思考整理のための3x3のフレームワークです。
「みんなのマンダラート」では、自分のマンダラートを作成し、他の人と共有できます。

## ✨ 主な機能(予定)

- 📊 **マンダラート作成**: Web上で直感的にマンダラートを作成
- 🌐 **投稿・共有**: 作成したマンダラートを公開
- 👀 **閲覧**: みんなのマンダラートを見て参考にできる
- 🎨 **シンプルUI**: 使いやすく、可愛いデザイン

## 🎯 プロジェクトの方針

- シンプルで分かりやすいUI/UX
- 低コストで運用可能な構成
- メンテナンスしやすいコード

## 📖 マンダラートとは?

マンダラートは、メジャーリーガーの大谷翔平選手も高校時代に使用していたことで有名な目標設定・思考整理の手法です。

1. 中央に「メインテーマ」を配置
2. 周囲8マスに「関連するテーマ」を配置
3. それぞれのテーマを中心とした新たな8マスを展開

合計81マス(9x9)で、目標達成のための具体的なアクションプランを可視化できます。
