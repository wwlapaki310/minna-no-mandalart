# 静的プロトタイプ - みんなのマンダラート

このディレクトリには、UI/UXを検証するための静的プロトタイプが含まれています。

## 🎨 デザインコンセプト

- **テーマ**: お正月 × 和 × シンプル
- **カラー**: 紅白 + 金色 + 松葉色
- **レスポンシブ**: スマホ・タブレット・PC完全対応

## 📂 ファイル構成

```
prototype/
├── index.html          # トップページ
├── create.html         # マンダラート作成画面
├── view.html           # マンダラート表示画面
├── list.html           # 一覧画面（未実装）
├── css/
│   ├── style.css       # 共通スタイル
│   ├── index.css       # トップページ専用
│   └── view.css        # 表示画面専用
└── js/
    ├── create.js       # 作成画面のロジック
    └── view.js         # 表示画面のロジック
```

## 🚀 使い方

### ローカルで確認

1. このディレクトリをブラウザで開く
2. `index.html` からスタート

### GitHub Pagesで公開（推奨）

1. GitHubリポジトリの Settings へ
2. Pages セクションで Source を `prototype/design` ブランチに設定
3. ディレクトリは `/prototype` を選択
4. 公開URLにアクセス

## ✨ 実装済み機能

### トップページ (`index.html`)
- ヒーローセクション
- 特徴紹介
- マンダラートの説明
- CTA

### 作成画面 (`create.html`)
- ✅ ガイド入力モード
  - Step 1: 中心目標入力
  - Step 2: 8つのテーマ入力
  - Step 3: 各テーマの具体的行動入力
- ✅ グリッド入力モード（9x9直接入力）
- ✅ リアルタイムプレビュー
- ✅ ローカルストレージ保存

### 表示画面 (`view.html`)
- ✅ 9x9マンダラート表示
- ✅ モバイル用セクション別表示
- ✅ シェア機能
- ✅ 画像保存機能（html2canvas使用）
- ✅ キーボードショートカット
  - `Ctrl/Cmd + S`: 画像保存
  - `Ctrl/Cmd + K`: シェア

## 📱 レスポンシブ対応

### デスクトップ (1920px以上)
- 9x9グリッド表示
- 横並びレイアウト

### タブレット (768px - 1919px)
- 調整されたグリッド
- 最適化されたスペーシング

### スマートフォン (〜767px)
- セクション別表示
- 縦スクロール最適化
- タッチ操作対応
